# Function Derivative Visualisation

The web application came from an inspiration that I had. 
To improve the understanding of the main funciton graphs and derivative function graphs. 

# INSTRUCTIONS: #

1. Go to the URL where this app is hosted.
2. Key in the Function to be graphed, the derivative function will appear.
3. Adjust the graph axes range for the graph to be featured properly. Do this for both functions.
4. Click a point on the main function curve, the x-coordinate and it's gradient value will be shown.
5. Click "Locked" button, to lock both the points on the main function and its derivative. 
6. Axes range are also transferred to the other graph once "Locked" button is clicked.
7. Use keyboard arrows or slide the points to move the point on the curve. Explore.
8. Users may download a copy of the index.html file to run the app offline.
